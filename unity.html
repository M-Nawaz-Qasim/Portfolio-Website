Flora Explorer - A Fun Fruit Collecting Adventure

Flora Explorer is a 2D adventure game where players collect different types of fruits while exploring a beautifully designed environment. The game features dynamic UI updates, smooth animations, and interactive character reactions.

Game Feature

  • Exciting Fruit Collection: Players collect specific fruits to complete levels.
  • Dynamic UI: The fruit counter updates in real-time as fruits are collected.
  • Challenging Levels: Players start with simple levels and unlock more difficult ones.
  • Character Reactions: The character displays happy or sad expressions based on correct or incorrect fruit collection.
  • Unlockable Levels: Completing a level unlocks the next one.
